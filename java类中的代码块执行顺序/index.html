代码块分类及其执行顺序

代码块分类

代码块，就是使用”{}”定义的一段代码，根据代码块定义的位置和关键字，可分为以下四种：

  • 普通代码块：定义在类的方法中的代码块。
  • 构造块：定义在类中的代码块（无修饰符）。
  • 静态代码块：使用static修饰的，在类中定义的代码块。
  • 同步代码块（与多线程有关，本文不涉及）
